Chat sample
Description: This sample application demonstrates usage of Brisleback server in chat application. Each written character is directly send to all chat users. You can see how user is writing message under nickname on users list. Data typed by user is sent to server, server processes data and broadcast it back to all logged in users. When new user logs in, all chat participants are notified, list is updated. Every written data is directly send to server, processed and send to all users, so you see every typed character.

Sources: github.com/bristleback/bristleback-samples

Scrum table sample
Description: Application presents simple table that can be used for managing scrum tasks. You can add new task, move it or resize. Each action sends event to server and all connected clients are notified. You can try it by opening two browser windows and adding new task. Tasks should appear in both windows. Also moving/resizing task will take place in both.

Sources: github.com/bristleback/bristleback-samples
