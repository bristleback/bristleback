Bristleback.template.TemplateController Class

Defined in: bristle-template.js:9
Module: template
Parent Module: Bristleback

Template controller is responsible for registering template frameworks and storing, caching and rendering templates. Before using template controller, template framework implementation must be registered via registerTemplateFramework() method.

Defined in bristle-template.js:88

Registers template framework that will be used by template controller. Template framework must provide 2 methods:

  • parseTemplate(templateName)
  • processTemplate(parsedTemplate, data)

Several template frameworks can be found in Bristleback.templateFrameworks object.
