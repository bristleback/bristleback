Bristleback.controller.Action Class

Defined in: bristle-controller.js:445
Module: controller
Parent Module: Bristleback

This is a single server action definition, created within a server action class. This class provides fluent API for building action behaviour. Constructor of this class should not be used directly by application user. User should rather create action using defineDefaultAction and defineAction .

Defined in bristle-controller.js:481

Sets a rendering response handler function for this action. If there is another response handler defined for this action (by invoking setResponseHandler() method), rendering handler function runs after that response handler, so normal handler can modify or prepare action response.

Parameters:

  • templateName String

    template name that should be used.

  • containerId String

    id of the parent container for rendered template.

  • rootObjectName String

    if specified, template controller will prepare literal object containing property with name given by this parameter and value returned by server.

  • renderingMode String

    name of rendering mode, one of the specified in Bristleback.templateController.renderingModes map. By default, "replace" mode is used.
