What is Bristleback?
Bristleback is a modern, complete WebSockets framework written in Java/JavaScript and powered by Spring Framework. The main goal of Bristleback is to bring creation of WebSockets applications into higher level of abstraction. Using Bristleback, you can forget about connection mechanism details and focus on your business logic only. With Bristleback, you will be pleasantly surprised how WebSockets technology becomes easy to use and productive. Bristleback framework consists of two parts:
  • WebSockets Server - written in Java, using several leading WebSockets protocol engines
  • WebSockets Client - written in JavaScript (Android client planned for next release)
Yet, while being high productivity framework, Bristleback still offers high performance results, which is achieved by caching meta information and using stable external WebSockets protocol libraries.
Main features:
  • Ability to create web and standalone applications
  • Jetty, Netty and Tomcat WebSockets engines support
  • Avoiding low level connection details, operating on high level abstractions
  • Message serialization working out of the box
  • Integrated with Spring Framework
  • Easy in usage and configuration
