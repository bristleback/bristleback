Bristleback.controller.ActionCallback Class

Defined in: bristle-controller.js:167
Module: controller
Parent Module: Bristleback

This class is the highest level response handler used in server actions. Action callbacks can be placed as last parameter in server action invocations. Action callback contains fields for both normal and exception responses.
